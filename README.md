# Diamond-Kata

Das Programm gibt das Alphabet bis zu gegebenen Buchstaben in Form eines Diamanten aus.
Falls 'E' der gesuchte Buchstabe wäre, sähe der Diamant wie folgt aus:

    ----A----
    ---B-B---
    --C---C--
    -D-----D-
    E-------E
    -D-----D-
    --C---C--
    ---B-B---
    ----A----
